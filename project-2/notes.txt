How to form a connection:
	1. Create a SERVER socket()
	2. Configure the SERVER socket using bind() and listen()
	3. Block the SERVER using accept() and wait for CLIENT to connect
	4. Create a CLIENT socket()
	5. Send a connection from the CLIENT to the SERVER using connect()
	6. Use send() and recv() to communicate between both sides
	7. Use close() to properly close and free the file descriptors the connection on both sides
